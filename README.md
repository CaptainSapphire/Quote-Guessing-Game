## How to make it yourself
Required Installations: [BeautifulSoup](https://pypi.org/project/beautifulsoup4/) (for webscraping), and python. 

You can Beautiful Soup with this command in your command prompt:
```
pip install beautifulsoup4
```
[See here for as tutorial, if you're having trouble with Beautiful Soup.](https://pytutorial.com/install-beautifulsoup-in-python-step-by-step/) <br>
